DILENDRA PANT PORTFOLIO WITH ARTICLES SECTION
=============================================

Directory Structure:
- index.html (Main portfolio page)
- css/style.css (All styles)
- js/main.js (All JavaScript functionality)
- articles/ (Technical articles directory)
  - index.html (Articles listing page)
  - article-*.html (Individual articles)
- img/ (Image assets)
- svg/ (Logo and icons)
- files/ (PDF and documents)

How to Use:

1. UNZIP the file to your web server directory
2. Open index.html in a browser
3. To add new articles:
   a. Create a new HTML file in /articles/ folder
   b. Follow the template from existing articles
   c. Add the article metadata to articles/index.html script section
   d. The article will automatically appear

Features:
- Expandable articles section
- Category filtering
- Full-screen article modal
- Responsive design
- Easy to add/edit articles
- Clean, professional design

To edit articles:
1. Open articles/index.html to modify the articles list
2. Edit individual article HTML files in /articles/ folder
3. Modify css/style.css for styling changes
4. Modify js/main.js for functionality changes
